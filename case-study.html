Case Study: Chat App

Overview

The Chat App is a mobile application that allows users to engage in real-time conversations, send image and audio messages, and share their current location. This app allowed me to explore mobile development with React Native, focusing on key features such as real-time chat, multimedia sharing, and data accessibility.

Challenge

To create a fully functional chat application for mobile devices, complete with user authentication, real-time data storage, and client-side data retention. This was my fifth project in the CareerFoundry program, providing a chance to deepen understanding of JavaScript applications and mobile app development.

Objective

To build a native mobile chat application in React Native, meeting requirements for:

  • Real-time messaging with a robust chat interface
  • User authentication and data management
  • Accessibility for screen readers
  • Offline functionality to access conversations when disconnected

In addition, I was tasked with completing five learning milestones within a one-month time frame.

Role

Full Stack Developer, with a focus on mobile development.

Project Scale

Completed in 21 days on a part-time basis. Required covering two learning milestones per week, which included:

  1. Building Native Applications with JavaScript
  2. Chat UIs & Accessibility
  3. Real-Time Applications & Data Storage
  4. Storing Data on the Client Side
  5. Communication Features
